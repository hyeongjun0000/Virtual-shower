Project Virtual Shower
1. 배경 및 목적
    1.1 개발 동기
       컴퓨터공학 특인지 샤워 안하는 사람이 꽤 있다.
       강의실에는 알 수 없는 식초향이 난다.
       게다가 현재 작성기준 여름 초입이라 이제 더워질건데 지금 날씨에서 샤워를 안하면 더 땀나고 더 냄새가 나게 된다..
       결국 참지 못하고 '컴공식 샤워' 프로그램 제작을 기획하게 되었다.
    1.2 문제 정의
        - 샤워하는걸 귀찮아 함
        - 샤워 할 줄 모름
        - 땀 및 노폐물이 쌓임
        - 냄새가 쌓임
        - 냄새가 퍼짐
    1.3 목적
        해당 프로그램으로 샤워 순서 방법을 공유하여 샤워 전 미리 방법을 공부할 수 있고 샤워의 중요성을 알 수 있게함.
        또한 샤워의 즐거움을 해당 프로그램을 통해 꺠달을 수 있으면 베스트
3. 제작 계획
    일단 내 샤워 방식을 참고하여 프로그램 제작에 사용
    2.1 내 샤워 순서도
        1. 물 on //샤워 전 행구기
        2. 물 off
        3. 샴푸
        4. 트리트먼트
        5. 바디 워시
        6. 물 on // 비눗거품 씻어내기
        7. 물 off
    2.2 개발 형태
        노트북 캠으로 전체화면 띄우고
        아래에 각 기능 별 버튼 : 물 샴푸 바디워시 환풍기 //트리트먼트는 구현이 애매할 듯?
        사용자 행동 분석 : 머리감기, 바디워시
        전체화면 흐림 정도 (%)

4. 개발 내용 및 추진 계획
    3.1 핵심 기능
        - 수도꼭지로 물 on/off
        - 샴푸로 머리감기
        - 바디워시로 몸 닦기
        - 환풍기로 습도 조절
    3.2 구현 기능
        아래에 버튼 (물 샴푸 바디워시 환풍기)
        물
        수도꼭지 켤 경우
        - 물 입자(작은 원모양)이 화면 위에서 아래로 떨어지며 사라짐.
        - (바 형식으로) 물 온도 조절 가능
        - IF 사용자 몸에 거품(샴푸 or 바디워시)이 있을경우 사라짐
        - 뜨거운 물로 조절될 경우 전체화면이 서서히 흐려짐.
        수도꼭지 끌 경우
        - 화면 흐려짐이 있을 경우 서서히 원상복귀

        환풍기
        - 화면이 일정 흐려진 정도를 넘을 경우 환풍기를 켜서 습도를 조절하라는 안내문장 표시.
        - 화면 흐려짐 현상을 '수도꼭지 끌 경우'보다 빠르게 원상복귀

        샴푸
        - 샴푸 버튼 누르면 머리를 감아보라는 문구 표시
        - 손으로 머리 감는 행위를 하면 머리에서 거품이 생김
        - 샴푸 머리 만들기 //국룰

        바디워시
        - 바디워시 버튼 누르면 바디워시 향 선택 (코튼 향/시트러스/머스크)
        - 손으로 몸을 닦는 행위를 하면 몸에 거품이 생김

5. 결과 및 기대효과
    - 사용자가 샤워에 대한 거부감 (↓)
    - 사용자가 샤워에 대한 재미 (↑)
    - 샤워의 중요성을 알게됨.

6. 참고 자료
